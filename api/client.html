Get slippy maps tile URL (e.g., /zoom/x/y.png).

Parameters

  • projection (str) ‚Äì The Proj projection to use for the tile layer. Default is EPSG:3857.

  • band (int) ‚Äì The band of the source raster to use (default in None to show RGB if available). Band indexing starts at 1. This can also be a list of integers to set which 3 bands to use for RGB.

  • palette (str) ‚Äì The name of the color palette from palettable or colormap from matplotlib to use when plotting a single band. Default is greyscale. If viewing a single band, a list of hex colors can be passed for a user-defined color palette.

  • vmin (float) ‚Äì The minimum value to use when colormapping the palette when plotting a single band.

  • vmax (float) ‚Äì The maximized value to use when colormapping the palette when plotting a single band.

  • nodata (float) ‚Äì The value from the band to use to interpret as not valid data.

  • scheme (str) ‚Äì This is either linear (the default) or discrete. If a palette is specified, linear uses a piecewise linear interpolation, and discrete uses exact colors from the palette with the range of the data mapped into the specified number of colors (e.g., a palette with two colors will split exactly halfway between the min and max values).

  • n_colors (int) ‚Äì The number (positive integer) of colors to discretize the matplotlib color palettes when used.

  • grid (bool) ‚Äì Show the outline of each tile. This is useful when debugging your tile viewer.

Generate an ipyleaflet TileLayer for the given TileClient.

Parameters

  • source (Union[pathlib.Path, str, TileClient]) ‚Äì The source of the tile layer. This can be a path on disk or an already open TileClient

  • port (int) ‚Äì The port on your host machine to use for the tile server (if creating a tileserver. This is ignored if a file path is given). This defaults to getting an available port.

  • debug (bool) ‚Äì Run the tile server in debug mode (if creating a tileserver. This is ignored if a file path is given).

  • projection (str) ‚Äì The Proj projection to use for the tile layer. Default is EPSG:3857.

  • band (int) ‚Äì The band of the source raster to use (default in None to show RGB if available). Band indexing starts at 1. This can also be a list of integers to set which 3 bands to use for RGB.

  • palette (str) ‚Äì The name of the color palette from palettable to use when plotting a single band. Default is greyscale.

  • vmin (float) ‚Äì The minimum value to use when colormapping the palette when plotting a single band.

  • vmax (float) ‚Äì The maximized value to use when colormapping the palette when plotting a single band.

  • nodata (float) ‚Äì The value from the band to use to interpret as not valid data.

  • scheme (str) ‚Äì This is either linear (the default) or discrete. If a palette is specified, linear uses a piecewise linear interpolation, and discrete uses exact colors from the palette with the range of the data mapped into the specified number of colors (e.g., a palette with two colors will split exactly halfway between the min and max values).

  • n_colors (int) ‚Äì The number (positive integer) of colors to discretize the matplotlib color palettes when used.

  • attribution (str) ‚Äì Attribution for the source raster. This defaults to a message about it being a local file.

  • **kwargs ‚Äì All additional keyword arguments are passed to ipyleaflet.TileLayer.
